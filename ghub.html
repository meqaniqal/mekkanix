Github Usage Instructions:

Starting out:

  1. Go into bash in terminal.
  2. Go to website git directory in terminal and IDE.

Cmd line cmds to set up github for greater use from cmd line:

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

Add a file:


git add hello.html

Commit, add instructions:

  • To commit current:

git commit -m "added more stuff to instructions"

git commit -am to commit all local to local changes and provide a msg thus, the 'm.'

Get current status:

git status (get info about current local/github relationship.

Update github repository at github.com:

git push (update github from local.)
(does not work in cmd line anymore
unless for password, you create a personal access token.
Or you can use the push icon in pycharm instead of the command line.

Reset:

git reset --hard origin/master (reset local to github.)

git reset --hard <commit> (<commit> is hash of the version you want to reset to.)

Branching:

git branch (Tells what branches exist and what branch is currently the one pointed to as the official branch.)

git checkout -b branchname (Make new branch called "branchname"

(Changes will be made now only to the branch pointed to by head.)

git checkout branchname (switch to branchname as official.)
  • This switches the local version to the current version of the branch as well,
    (local version, I'd guess.)

Switch to master branch:

git merge branchname (this will merge the changes from the specified branchname with the current branch "master."
(Merge conflicts can occur.)

Creating and using other repositories:

If you add another repo from github, you can drag/drop a folder of your choice into it.
In terminal, you can go to that folder:

git init
git commit -m "(your description of what this version is about)"

now git push will push it into that repo.
